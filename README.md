Sales & Customer Analytics Dashboard 2023
🔹 Project Overview
This project presents an interactive Sales and Customer Analytics Dashboard designed to provide meaningful business insights for the year 2023.
It includes a comparison with the previous year (2022) to highlight growth, trends, and performance improvements.

🔹 Features
Interactive and user-friendly dashboard

Year-over-Year (2023 vs 2022) comparison

Sales, Profit, Quantity, and Order analysis

Customer distribution based on number of orders

Top 10 customers by profit

Sub-category level performance analysis

Sales & Profit trends over time

🔹 Dataset Information
The dataset used in this project is stored in the data folder and includes:

orders.csv – Order details

customers.csv – Customer information

products.csv – Product details

sales.csv – Sales transactions
